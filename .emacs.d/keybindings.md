# Emacs Keybindings Reference

## Navigation & Buffers

| Key | Command | Description |
|-----|---------|-------------|
| `C-x b` | consult-buffer | Switch buffer with preview |
| `C-x C-r` | consult-recent-file | Open recent file |
| `M-y` | consult-yank-pop | Browse kill ring |
| `M-i` | imenu | Jump to symbol in buffer |
| `M-o` | other-window | Switch to other window |
| `M-<arrows>` | windmove | Move between windows by direction |

## Search

| Key | Command | Description |
|-----|---------|-------------|
| `M-s l` | consult-line | Search lines in buffer |
| `M-s g` | consult-grep | Grep across files |

## Projects

| Key | Command | Description |
|-----|---------|-------------|
| `C-x p p` | my-projects-switch | Switch project (saves/restores files) |
| `C-x p a` | my-projects-add-dir | Register a directory as a project |
| `C-x p f` | project-find-file | Find file in current project |

## Git

| Key | Command | Description |
|-----|---------|-------------|
| `C-x g` | magit-status | Open Magit status |

## Buffer Management

| Key | Command | Description |
|-----|---------|-------------|
| `C-x C-b` | ibuffer | List buffers (remapped from list-buffers) |

## macOS

| Modifier | Mapping |
|----------|---------|
| Command | Meta (`M-`) |
| Option | Disabled as modifier |
